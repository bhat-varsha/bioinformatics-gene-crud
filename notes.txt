Express.js is a web framework built on top of Node.js.
Express provides ready-made tools for common backend tasks.
Express is similar to Flask.Both create endpoints.same concept diff language

One of Express's biggest features.
MIDDLEWARE:Middleware is code that runs before reaching the final route.

| Flask        | Express    |
| ------------ | ---------- |
| Python       | JavaScript |
| @app.route() | app.get()  |
| request      | req        |
| response     | res        |
| jsonify()    | res.json() |

Node.js provides the runtime environment, 
Express.js provides a framework that simplifies routing,request handling,middleware management,API development
